DealFinder

DealFinder is a Windows desktop application that uses the official eBay Browse API to perform structured product discovery searches.

Overview

  • Desktop-based (not a hosted web service)
  • Uses official eBay Browse API endpoints
  • Implements rate limiting and backoff handling
  • Monitors daily API usage
