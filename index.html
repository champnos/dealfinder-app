DealFinder

DealFinder is a Windows desktop application that uses the official eBay Browse API to perform structured product discovery and listing analysis.

Overview

  • Windows desktop application (not a hosted web service)
  • Uses official eBay Browse API endpoints only
  • Structured search profiles and keyword logic
  • Rate limit handling with exponential backoff
  • Daily API usage monitoring
